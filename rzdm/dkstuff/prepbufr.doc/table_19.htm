Table 19.  Code table for PREPBUFR report types used by upper-air Real-Time Mesoscale (GSI) Analysis (RTMA) and UnRestricted Mesoscale Analysis (URMA) (last revised 4/17/2020- (no more updating for past events expected unless something important later found to be missed or incorrect).
 

Key:

     In the "Report Type" column, a  GREEN report type means that report type can be found in the PREPBUFR file.  A RED report type means that report type is currently never found in the PREPBUFR file.  An "R" after the report type means this type is restricted with respect to redistribution outside of NCEP for some amount of time.

     In the "Description" and "Note" columns, observation types are designated as follows:
          cldch - cloud ceiling height (derived from cloud amount and cloud height)
          gust - 10-m gust
          howv - significant wave height
          mitm - daily minimum temperature
          mxtm - daily maximum temperature
          Pstn - station (surface) pressure
          PWl - layer precipitable water
          PWt - total column precipitable water
          q - specific humidity
          rr - rain rate
          spd
          tcamt - total cloud amount
          Tb - brightness temperature
          Ts - sensible (dry bulb) temperature
          Tv - virtual temperature
          uv - u- and v-components of wind
          vis - surface horizontal visibility
          wdir - wind direction
          wspd - wind speed
          wspd10m - wind speed 10 meters above surface
          z - vertical coordinate height

     In the "Note" column, RTMA and URMA runs are designated as follows:
          AK3p0 - 3.0 km RTMA and URMA Alaska run
          CO2p5 - 2.5 km RTMA and URMA CONUS run
          CO2p5u - 2.5 km URMA CONUS run
          RU2p5 - 2.5 km RTMA CONUS Rapid-Update run
          GU2p5 - 2.5 km RTMA Guam run
          HI2p5 - 2.5 km RTMA and URMA Hawaii run
          PR2p5 - 2.5 km RTMA and URMA Puerto Rico run
          ALL - all RTMA and URMA runs
          ALLu - all URMA runs

     In the "Note" column, GREEN lines refer to observation types that end up being used by the assimilating analysis, BLUE lines refer to observation types that end up being monitored (but not actually used) by the assimilating analysis, and RED lines refer to observation types (or entire reports) that are not used by the assimilating analysis due to the reason(s) given (they may, however, be monitored). BLACK lines refer to informational material.

Note: The RTMA and URMA both employ a station reject list for all types of data below.  One part of this list is compiled every month by the Weather Foreacast Offices and the other part is generated dynamically at every hour by the RTMA and URMA.
