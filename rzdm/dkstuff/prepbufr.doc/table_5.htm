Table 5.  Code table for PREPBUFR report types used by Regional Rapid Refresh (RAP) GSI analysis (last revised 3/19/2018- (no more updating for past events expected unless something important later found to be missed or incorrect).  [Note this does not denote PREPBUFR report types used by High-Resolution Rapid Refresh (HRRR) GSI analysis.]
(Note: see Table 21 for code table referring to pseudo-PREPBUFR aircraft profile report types.)
 

Key:

     In the "Report Type" column, a  GREEN report type means that report type can be found in the PREPBUFR file.  A RED report type means that report type is currently never found in the PREPBUFR file.  An "R" after the report type means this type is restricted with respect to redistribution outside of NCEP for some amount of time.

     In the "Description" and "Note" columns, observation types are designated as follows:
          gos_ctp -  GOES cloud-top pressure, total cloud cover, cloud-top temperature
          mta_cld - METAR cloud amount, height of cloud base, present weather, visibility, dewpoint
          p - vertical coordinate pressure
          Pstn - station (surface) pressure
          PWl - layer precipitable water
          PWt - total column precipitable water
          q - specific humidity
          rr - rain rate
          sst - sea-surface temperature
          Tb - brightness temperature
          Td - dewpoint temperature
          Ts - sensible (dry bulb) temperature
          Tv - virtual temperature
          u, v - u- and v-components of wind
          wspd - wind speed
          z - vertical coordinate height
 
     In the "Note" column, GREEN lines refer to observation types that end up being used by the assimilating analysis, BLUE lines refer to observation types that end up being monitored (but not actually used) by the assimilating analysis, and RED lines refer to observation types (or entire reports) that are not used by the assimilating analysis due to the reason(s) given (they may, however, be monitored). BLACK lines refer to informational material.
