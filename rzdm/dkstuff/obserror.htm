Observation type 120 is a rawinsonde (mass field data only). 

The columns

120 OBSERVATION TYPE
 0.11000E+04 0.12000E+01 0.20000E+01 0.10000E+10 0.10000E+01 0.10000E+10
 0.10500E+04 0.12000E+01 0.20000E+01 0.10000E+10 0.10000E+01 0.10000E+10
 0.10000E+04 0.12000E+01 0.20000E+01 0.10000E+10 0.10000E+01 0.10000E+10

have the following meaning
    column 1:  pressure level in hPa
    column 2:  temperature observation error (K)
    column 3:  moisture (RH) error in units of tens of  % RH  (e.g., 0.20000E+01 = 20% RH error)
    column 4:  wind error (m/s)
    column 5:  surface pressure (hPa)
    column 6:  total precipitable water (mm)

So for the above example, the assumed obs errors for sonde data at 1000 hPa are
   temperature:  1.2 K
   moisture:  20% RH
   surface pressure:  1 hPa

Rawinsonde winds have observation type 220.

Thus, the assigned error for sonde winds at 1000 hPa is 1.4 m/s

A value of  0.10000E+10 indicates missing.
