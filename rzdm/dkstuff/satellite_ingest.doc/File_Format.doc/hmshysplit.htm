Format for NESDIS Hazard Mapping System (HMS) HYbrid Single-Particle Lagrangian Integrated Trajectory (HYSPLIT) preliminary and final model smoke text files

 

A comma delimited file.  The first (header) line defines the contents in each of the 5 columns in subsequent lines.  Each subseqent line defines the location [latitude (N+,S-) and longitude (0-180 E+, W-) to nearest 0.001 degree], date-time (YYYYMMDD hhmm, UTC) and duration (hhmm) of each fire/smoke event.

For example:

-- start of file contents --
     Lon,    Lat, YearMmDd, HhMm,  Dur
 -81.314, 27.035, 20100216, 1345, 0500
 -81.256, 27.108, 20100216, 1415, 0300
 -97.335, 27.580, 20100216, 1845, 0600
 -97.335, 27.580, 20100216, 1845, 0400
 -95.400, 28.899, 20100216, 2015, 0200
 -94.642, 29.584, 20100216, 1915, 0500
 -94.653, 29.585, 20100216, 1915, 0300
 -93.027, 29.786, 20100216, 2145, 0100
-- end of file contents --
